Test for validity and reliability

There are two items to submit this week

  1. Simulate a “large” data set1 using whatever software package you prefer. Below are some links to help you out for the software packages2

    • Excel

    • Qualtrics

    • R (contact me because there are many like this)

  2. To assess validity, use whatever software you wish to conduct an exploratory factor analysis of your data. For those who use R, take a look at the EFA walkthrough.

  3. To assess reliability, use whatever software you wish to perform a reliability analysis. For those who use R, take a look at the Internal Consistency walkthrough.

  4. In a single page, describe what methods you used to assess both validity3 and reliability.

Submit this single page AND a copy of your simulated data set to eCampus.

The entire sequence leading up to the final proposal can be found on the main Tasks page.
