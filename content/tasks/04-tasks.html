Refine the Draft

Like last week, this is a also a two step-process:

  1. Refine your instrument if needed1

  2. In a word file, create a five column table like so

index item measure group changes
          where
          a. index is an item number given in order of appearance.
          b. item is a statement or question.
          c. measure is the associated measure.
          d. group is the associated group of items, if applicable.
          e. changes is a brief description of any alterations performed, if applicable.
