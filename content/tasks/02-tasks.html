Creating an Argument

Submit a one to three page proposal outlining specific questions that you would like to assess and then describe an instrument that would be used to assess those items. Include general details on what measures would be used. Submit this document using APA 7th Edition formatting to eCampus.

The entire sequence leading up to the final proposal can be found on the main Tasks page.
