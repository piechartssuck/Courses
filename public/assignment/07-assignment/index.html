A Map to the Past

  • Read the Submission Directions

Read the Submission Directions

Please submit a PDF out using an Rmarkdown file for this task1. Please submit a PDF using an Rmarkdown file for this task2 both to eCampus and Slack.

You may use the data set provided in this week’s walkthrough or any other valid set with citations to address the following criteria regarding the 2016 general election.

For each election year, do the following:

  1. The differences between the popular and electoral college votes by census tract, county, and state. For each, show both the flat standard representation and where votes counts and/or percentages are displayed by size. For a example of a state level representation of this, please look here.

  2. How differing demographical subgroups voted. You should include at minimum age, race and sex3 and at least three others of your choice4.

  3. A single final visualization showing the differences in the popular and electoral college votes by census tract, county, and state.
    Please submit your output to both

    • eCampus and

    • Slack a PDF and/or HTML output using an Rmarkdown file for this task.

  4. Bonus! Display state5 elections by census tract and county.

:::

  1. If you use an external file for your data, please submit that as well.↩︎

  2. If you use an external file for your data, please submit that as well.↩︎

  3. Please use the applicable U.S Census standard for these demographics.↩︎

  4. Need not be a criteria reported in the U.S. Census↩︎

  5. of your choice↩︎
